# 要求定義書：期間限定投稿SNS「Lifespan Feed（仮）」

## 1. プロジェクト概要
本プロジェクトは、SNSにおける「情報の蓄積」による心理的負荷を軽減し、「今、この瞬間」のコミュニケーションを最大化することを目的とする。投稿に「寿命（自動削除機能）」を設けることで、ユーザーの自己開示に対する心理的障壁を下げる。

---

## 2. 業務要求
ユーザーは、投稿の性質（一時的な感情、日常の共有、議論など）に応じて、適切な公開期間を選択し、期限が過ぎた投稿はシステムによって自動的に消去される。

---

## 3. 機能要求

### 3.1 寿命選択機能
投稿時に、以下の3つのカテゴリーに基づいた保持期間を選択できること。

| カテゴリー | 選択可能な時間 | ユーザー体験の意図 |
| :--- | :--- | :--- |
| **1. 感情のデトックス** | 15分、30分、60分 | 言い放ちたい感情の整理、一時的な独り言。 |
| **2. インスタント共有** | 24時間 | ストーリーズ形式の日常報告。 |
| **3. 議論・FB** | 48時間、72時間 | フィードバックの収集、特定のテーマでの議論。 |

### 3.2 投稿管理・表示機能
* **カウントダウン表示:** 各投稿には「作成日時」ではなく「残り時間（例：残り12分）」を表示する。
* **自動削除プロセス:** * 保持期間終了後、即座にタイムラインから非表示とする。
    * データベースからの物理削除、または一般ユーザーからアクセス不能なアーカイブ状態への移行を行う。

### 3.3 通知機能
* 「議論・FB」カテゴリーにおいて、終了1時間前に「間もなく終了します」といった通知を送るオプション。

---

## 4. 非機能要求

### 4.1 信頼性・正確性
* **削除の定時性:** 設定された寿命から5分以上のラグなく削除処理が実行されること。

### 4.2 セキュリティ
* 有効期限が切れたURLへの直接アクセスに対し、404エラーまたは「この投稿は寿命を迎えました」というメッセージを返すこと。

---

## 5. 補足事項（エビデンスおよび不確定要素）

> **不確定な情報を含むことの明示：**
> 「感情のデトックス」として短時間の削除機能を提供することが、ユーザーのメンタルヘルス改善に直接寄与するという確定的な科学的エビデンスは現時点で十分に確立されていません。
> ただし、エフェメラル（短命な）SNSの研究（例：Snapchat等）においては、「完璧主義からの解放」や「プライバシー感の向上」が報告されています。本システムにおけるデトックス効果の検証は、リリース後のユーザー動向調査（定性・定量）に依存します。
