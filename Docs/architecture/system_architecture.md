# システムアーキテクチャ

本ドキュメントでは、Kieru SNS App の技術スタックおよびシステム構成について定義します。

## 1. 技術スタック

### フロントエンド

| 項目 | 採用技術 | 備考 |
| :--- | :--- | :--- |
| UIフレームワーク | React | コンポーネントベースのUI開発 |
| ビルドツール | Vite | 高速な開発環境とビルド |
| 言語 | TypeScript | 静的型付けによる保守性の向上 |
| スタイリング | Tailwind CSS | ユーティリティファーストなCSSフレームワーク |
| データフェッチ | TanStack Query | 非同期データの取得・キャッシュ・同期 |
| UIコンポーネント | Shadcn UI | 高品質でカスタマイズ可能なコンポーネント群（推奨） |

### バックエンド（暫定）

| 項目 | 採用技術 | 備考 |
| :--- | :--- | :--- |
| フレームワーク | .NET | 高いパフォーマンスと型安全性 |
| 言語 | C# | 堅牢なバックエンド開発 |

### データベース

| 項目 | 採用技術 | 備考 |
| :--- | :--- | :--- |
| RDBMS | SQL Server | 信頼性の高いデータ管理 |
| コンテナ | Docker | 環境の統一化 |
