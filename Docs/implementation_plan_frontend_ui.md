# 実装計画：フロントエンドUIプロトタイプの実装

## 1. 目的
フロントエンドの主要なUIを構築し、ユーザー体験（残り時間の可視化、情報の儚さの演出）を早期に検証する。

## 2. タスク概要
1.  **デザインシステムの設定**
    *   Tailwind CSS のカラーパレット、フォントの設定。
    *   グラスモーフィズム用のアセット/スタイルの定義。
2.  **モックデータによるプロトタイピング**
    *   バックエンドの実装を待たずにUIを確認できるよう、一時的なダミーデータを使用。
3.  **主要コンポーネントの構築**
    *   タイムライン、投稿カード、寿命選択UI。

## 3. 詳細手順

### Step 1: スタイリング定義 (`index.css`)
*   ダークモードを基調とした背景色の設定。
*   カードのブラー（ぼかし）効果の定義。

### Step 2: 投稿カードコンポーネント (`PostCard.tsx`)
*   残り時間バッジのロジック（時間による色の変化）を実装。

### Step 3: 投稿作成画面 (`CreatePost.tsx`)
*   3つのカテゴリー（デトックス、共有、議論）を切り替えるタブUIを実装。

---

## 4. 進捗確認用成果物（モックアップ）
添付の「kieru_sns_ui_mockup」のような、視覚的に訴求力の高いUIをReactで再現します。
