# Issue: フロントエンドUIプロトタイプの実装

## 概要
外部設計（ワイヤーフレーム）に基づき、フロントエンドの主要画面（タイムライン、投稿作成）の実装を行います。ユーザーが「情報の儚さ」を視覚的に体験できる、モダンでプレミアムなUIを目指します。

## 開発詳細
1. **フロントエンド環境構築の最終確認**
    - Vite + React + Tailwind CSS の動作確認。
2. **共通コンポーネントの実装**
    - `CountdownBadge`: 残り時間に応じて色や表現が変わるバッジ。
    - `PostCard`: グラスモーフィズムを採用した投稿表示カード。
3. **主要画面の実装**
    - `TimelinePage`: 投稿一覧画面。
    - `CreatePostModal`: 寿命選択機能付きの投稿作成画面。
4. **デザイン調整**
    - ダークモード、アニメーション（投稿のフェードアウト等）の実装。

## 完了条件
- [ ] タイミングよく投稿が表示され、残り時間がリアルタイムで更新されること。
- [ ] 投稿作成時に寿命が選択でき、プレビューができること。
- [ ] モックアップに近いデザインがReact上で再現されていること。

## 備考
- 初期段階ではバックエンドをモック（mswや静的データ）で代用し、UI/UXの検証を優先する。
